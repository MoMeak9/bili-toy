# 本地音频实验室

本地音频实验室是一款开箱即用的单页 Web 音频工具：打开页面即可上传音频或打开示例，在浏览器本地查看波形、播放、套用变声预设、调整参数，并导出 WAV。

## 快速开始

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```

## MVP 功能

- 上传本地音频或打开内置示例音频
- 波形展示与 A/B 对比，A 为原始音频，B 为处理后音频
- 播放 / 暂停、进度时间显示
- 机器人、魔鬼低音、松鼠音、电话音、广播主持五个 P0 预设
- 音量、变速、变调、淡入、淡出参数调整
- WAV 整段导出与本地下载
- 桌面端快捷键：`Space` 播放 / 暂停，`?` 打开快捷键弹窗
- 桌面三栏布局，移动端底部标签面板

## 本地与隐私

应用不需要账号、不上传音频、不依赖后端接口，也不使用 AI。音频解析、播放、效果处理、离线渲染和 WAV 编码都在浏览器本地完成。

## 技术说明

- React + Vite + TypeScript 构建单页应用
- Tone.js 负责播放、效果链和离线渲染
- wavesurfer.js 负责波形显示
- Zustand 管理纯数据状态
- 自写 WAV 编码器用于导出
