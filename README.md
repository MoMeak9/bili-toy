# 本地音频实验室

本地音频实验室是一款开箱即用的单页 Web 音频工具：打开页面即可上传音频、录制麦克风或打开示例，在浏览器本地查看波形、播放、套用变声预设、查看分析数据，并导出 WAV / MP3。

## 快速开始

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```

## 功能

- 上传本地音频、使用麦克风录音，或打开内置示例音频
- 波形展示与 A/B 对比，A 为原始音频，B 为处理后音频
- 播放 / 暂停、进度时间显示
- P0 + V1.5 预设：机器人、魔鬼低音、松鼠音、电话音、广播主持、外星人、磁带机、山洞回声、8-bit
- 效果链面板展示当前预设的处理步骤
- 音量、变速、变调、淡入、淡出参数调整
- 基础分析面板：时长、采样率、声道、Peak、RMS 与懒加载频谱图
- WAV / MP3 整段导出与本地下载
- 桌面端快捷键：`Space` 播放 / 暂停，`?` 打开快捷键弹窗
- 桌面三栏布局，移动端底部预设 / 效果 / 参数 / 分析标签面板

## 本地与隐私

应用不需要账号、不上传音频、不依赖后端接口，也不使用 AI。音频解析、麦克风录制、播放、效果处理、离线渲染、频谱分析和 WAV / MP3 编码都在浏览器本地完成。

## 技术说明

- React + Vite + TypeScript 构建单页应用
- Tone.js 负责播放、效果链和离线渲染，并通过动态导入降低首屏包大小
- wavesurfer.js 负责波形显示，并在进入编辑器后懒加载
- Zustand 管理纯数据状态
- 自写 WAV 编码器与懒加载 MP3 编码器用于导出
